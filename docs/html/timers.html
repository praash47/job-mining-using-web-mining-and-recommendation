timers.jobextractor.extract_jobs(pending_jobs)[source]¶

Extracts job parameters from the pending jobs in a cyclic fashion. Cyclic fashion means each job website gets once per cycle for it’s single job extraction.

timers.mainloop module¶

This file is responsible for looping through the main operations every day. This is the main entrypoint for the backend section and signifies the general flow of program seperate from Django.

Module contents¶

Timers Module is responsible for timely and ethical mining of job websites by timely seperating the requests. It handles three core functions at startup and in 12.00 am everyday: 1. Filtering deadline expired jobs 2. Checking for new Job websites and it’s URLs 3. Checking for new Job URLs in existing job websites
