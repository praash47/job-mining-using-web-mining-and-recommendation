recommender.utils module¶

Utilities reqeuired for the views, or in dealing with the client.

Consists: * decode_utf(body): decodes utf and returns the body dict. * jaccard_similarity(job_skills, candidate_skills): computes the Jaccard Similarity and returns the result. * get_skills_from_string(skills_string, brackets): Gets skills from skill string by removing the brackets. * lies_in_salary_range(job_salary, salary): Returns true if the salary lies in range of job_salary. * deadline_left(job_deadline, deadline): Returns true if the deadline lies within job_deadline. * get_deadline(date): Returns days and months from given deadline date string.
